Vision & Roadmap

Graph BI

Qwiery Dashboards are a way to visualize data in a more graphical way. They are a collection of widgets that can be arranged in a grid. The widgets can be of various types, such as charts, tables, and maps.

Plugins

Use plugins to assemble functionality or to extend Qwiery, see e.g. the semantic extensions or the schema and ontology plugin. Creating your own plugins is simple, see our plugins guide.

Storage Adapters

Use your favorite database to store and access graphs: Neo4j, AWS Neptune, MySql, ...
